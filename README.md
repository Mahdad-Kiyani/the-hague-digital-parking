# The Hague Digital Parking

This repository documents the large-scale parking system modernization program for the City of The Hague, where I served as Technical Scrum Master and Developer between 2015 and 2021. The project replaced legacy permit workflows with license plate recognition, API-driven enforcement, and AI-based anomaly detection.

## 🧰 Tech Stack

- Java, AS-400, HTML5/JS, Redux
- REST APIs (DigiD, CityPermit, RDW)
- IBM iSeries, PostGIS, Snowflake
- Azure DevOps, GitHub Actions, Terraform
- Power BI, OAuth2, Loop-App
- DevSecOps, GitOps, BISLnext

## ⚙️ Key Contributions

- Built REST APIs bridging DigiD, RDW, and CityPermit systems
- Refactored parking portal frontend/backend to modular microservices
- Designed state machines for permit lifecycle management
- Engineered VPN/SFTP integrations for municipal and SaaS vendors
- Led Agile sprints, DevSecOps loops, and CI/CD deployments

## 📊 Data & Audit Engineering

- Real-time scan data ingestion with IBM AS-400 integration
- Built Power BI dashboards tracking anomalies and lifecycle KPIs
- Implemented Snowflake Time Travel for auditing
- Applied role-based access and GDPR-compliant storage

## 📈 Outcome

- 60% reduction in permit processing latency
- Fully decommissioned paper workflows
- Achieved 99.99% uptime with fault-tolerant architecture
- Increased enforcement transparency and traceability
